# なぜ風は帯状に吹くのか — コリオリ力とポテンシャル渦度

偏西風はなぜ東西方向に帯状に吹くのか？黒潮はなぜ蛇行するのか？
その答えは、地球の自転が生み出す「コリオリ力」と「ポテンシャル渦度（PV）の保存」にあります。

この動画では、メリーゴーランドの例えからコリオリ力を直感的に理解し、PV = f/h という簡潔な保存則が偏西風の帯状構造、海嶺による海流の偏向、黒潮の蛇行、さらには木星の大赤斑まで説明できることを解説します。

🔍 この動画でわかること：
・ コリオリ力とは何か — メリーゴーランドの例えで直感的に理解
・ コリオリパラメーター f が緯度によって変わる理由
・ ポテンシャル渦度 PV = f/h の保存則
・ 偏西風が帯状に吹く理由
・ 海底地形（海嶺・海溝）が海流を曲げるメカニズム
・ 黒潮の蛇行と沖縄トラフの役割
