### §1. フック（イントロダクション）

紅茶をかき混ぜたときの茶葉の動きや、台風の衛星写真、洗面台の排水溝の渦。
日常のいろんな場所で「渦」を見かけますよね。

この渦の物理を突き詰めていくと、地球の大気や海の流れ、さらには木星の大赤斑のような壮大なスケールの流れや渦の秘密を解き明かす理論にたどり着くんです。

というわけで、今日は渦の物理についてお話しします。

---

### §2. 渦度とは何か

まず、渦を理解するために流れの回転を考えていきます。

水の流れの中に小さな正方形のかけらを置いたと考えましょう。

正方形の辺に沿って、速度をぐるっと一周分足し合わせると、
この正方形が流れから受ける回転の強さの総量になります。
さらにこの回転の総量を正方形の面積で割った量を渦度と呼びます。
イメージとしては正方形のかけらをどんどん小さくしていった時にかけらが受ける回転の強さを表す量が渦度と言えます。

```latex
% テロップで補足
「足し合わせる」は厳密には線積分
```

2次元の場合、渦度はこのような式で表されます。

```latex
% Manim アニメーションで表示
ω = ∂v/∂x − ∂u/∂y
```

ここでuとvは、それぞれ横方向と縦方向の流れの速さです。
つまり渦度とは、「その場所で流体がどれだけ回転しているか」を表すんですね。

### §3. 理想流体とエンストロフィ保存

今日の動画で考える流体はすべて「理想流体」とします。
理想流体とは、一言で言えば粘性がない流体のことです。

プールで前に進むと水が押し返してきますよね。
この押し返す力は水の粘性によって生じます。
粘性とは簡単に言えば、水や空気のような流体がどれだけネバネバしているかを表す量です。

理想流体ではそのような抵抗がありません。
ものすごくサラサラしている水のようにイメージしてもらえればOKです。

この理想流体の世界では面白いことが起こります。

大きな渦が小さな渦に分裂していっても、「渦の濃さ」の総量は変わらないんです。
小さな渦がたくさんあった方が全体ではより渦が激しくグルグルしているように見えますが、実はそうではないんですね。

この「渦の濃さ」を定量化した量を「エンストロフィ」と呼びます。
ちょっと聞きなれない量ですが、渦を理解するためにとても重要な概念です。

式で書くとこのようになります。
渦度の2乗を、積分して2で割った量ですね。

```latex
% Manim アニメーションで表示
Z = ½∫ω² dA
```

そして2次元の非圧縮理想流体では、新たな渦を産んだり消したりする外からの力が加わらない限りこのエンストロフィZが保存されます。
非圧縮とは空気のように膨らんだり縮んだりせず、水のように伸び縮みせず密度が一定、という意味ですね。

```latex
% テロップで補足
2次元・非圧縮・外力（渦度源）なし・適切な境界条件が必要
```

渦が分裂しても、合体しても、エンストロフィの合計は変わらない。
エネルギーや運動量のように、渦を表すエンストロフィも流れを理解する時にはとても重要な保存量なんですね。

エンストロフィが保存するのは粘性がない理想流体だから、ということには注意しましょう。
紅茶をかき混ぜても時間が経てば渦が消えてしまうように、現実の流体には粘性があるので、いつまでもエンストロフィが保存するわけではないんです。

### §4. 水面が動く現実 — 浅水波方程式

ここまでは水面の高さが一定の世界を考えてきました。
いわば「蓋のある水槽」のような、きれいな2次元の世界です。

でも、現実は海やプールのように蓋のない流体の運動をどこでも見ることができます。
蓋のない海やプールでは水面は上下しますよね。
つまり、水面が高い場所と低い場所で波ができます。

ここで登場するのが「浅水波近似」という考え方です。
水深に比べて水平方向のスケールが十分に大きいなら、
深さ方向のことは無視してしまおう、という考え方ですね。

浅水波方程式は、2つの式で構成されます。

1つ目は「連続の式」。これは質量保存を表します。
言い換えれば水は急に現れたり消えたりしない、ということを意味しています。

```latex
% Manim アニメーションで表示
∂h/∂t + ∇·(h**u**) = 0
```

2つ目は「運動量方程式」。
水面の傾きが、流れを生む、ということを意味しています。
平たく言えば水は高いところから低いところに流れる、という意味ですね。

```latex
% Manim アニメーションで表示
∂**u**/∂t + (**u**·∇)**u** = −g∇h
```

ここで大事なのは、水深hが変わるということ。
つまり渦が存在する場所の厚みが変わるわけです。
このとき、さっき見たエンストロフィの保存はどうなるでしょうか？

### §5. ポテンシャル渦度

水深が変わると渦度やエンストロフィそのものの保存は成り立たなくなります。

でも、代わりに「渦度を水深で割った量」が保存されるようになるんです。
この量のことを「ポテンシャル渦度」と呼びます。
英語で言うとPotential Vorticity、頭文字をとって略してPVと呼んでいきますね。

数式で書くとこうなります。

```latex
% Manim アニメーションで表示
PV = ζ / h
```

ζは相対渦度、hは水深です。
つまり渦度そのものではなく、高さあたりの渦の強さが保存量になるんですね。

直感的な理解のためにピザ生地を回すことを考えましょう。

生地を薄く伸ばすと、回転はゆっくりになります。
逆に、厚くまとめると、回転は速くなります。

これがPVの本質です。
水深hが変わっても、PVは一定に保たれる。
水深が浅い場所では渦の回転がゆっくりになり、水深が深い場所では渦の回転が速くなりますが、高さあたりの回転は実は一定なんです。

プールの水面のような波の高さが変化する流れの渦はPVの保存則によって理解することができるんです。

プールやピザの生地の回転は渦度と水深を考えるだけで十分です。
でも、海の流れや風の流れのような地球スケールの現象になると、別の力の影響も考えなければなりません。
それは地球の自転の影響です。

回転する物体の上で運動する物体は、その回転の影響を受けます。
この力のことをコリオリ力と呼び、回転している地球の上を動く海の流れや風の流れはコリオリ力の影響を受けるんです。

地球の自転の影響を考慮した時、コリオリパラメーターfを加えて、PVはこのように書き換える必要があります。

```latex
% Manim アニメーションで表示
PV = (ζ + f) / h
```

コリオリパラメーターfは緯度によって変わります。
北極や南極では、地球の自転軸が地面に対してほぼ垂直になっています。
そのため、水平方向に動く流体は自転の影響をフルに受けて、fは最も大きくなります。
一方、赤道に近づくと自転軸が地面と平行に近づいていくので、水平方向の運動に対する自転の影響が弱くなり、fは小さくなっていきます。
赤道ではfはゼロになるんです。
だからコリオリパラメーターは北半球では南から北に流れると大きくなり、逆に北から南に流れると小さくなるんです。

水深hが一定の場合、海の流れが南から北に流れると、コリオリパラメーターが大きくなり、
そのぶん相対渦度ζが小さくなる。つまり回転による渦が小さな流れになっていくんですね。
逆に北から南に流れる場合はその逆になり、渦は強くなると言えるんです。

水深ではなく渦度が一定の場合はどうなるか。
南から北に流れた時、コリオリパラメーターが増加する分、水深hも大きくなります。
逆に北から南に流れた場合には水深は浅くなるんです。

実際には地形や温度などの様々な条件によって渦度と水深の両方が変化するのですが、
PVが一定であるという事実はとても重要です。

そしてこれは水だけの話ではありません。
大気でも同じで、大気の厚みがhの役割を果たし、風の流れや渦を説明する理論になるんです。

この理論は1930年代から40年代にスウェーデンの気象学者カール＝グスタフ・ロスビーによって研究されました。
ロスビーをはじめとした当時の地球物理学者たちはPVにコリオリパラメーターを加えることで地球規模の風や海の流れを説明する理論を作り上げたんです。

この理論は地球規模の風や海流の研究に大きな影響を与え、
現在の気象予報や海洋予測でも重要な役割を果たしています。
黒潮の流れや、偏西風、台風などの様々な地球規模の流れの運動の研究は、
ロスビーたちの理論をもとに発展しています。

木星の大赤斑のような地球よりもはるかに大きな渦の運動も、
コリオリパラメーターを考慮したPVの保存をさらに発展させた理論によって理解が進められているんです。

