### §1. フック（イントロダクション）

偏西風は、地球をぐるっと一周するように西から東へ帯状に吹いています。
台風は北半球では反時計回りに渦を巻いています。
木星の大赤斑は300年以上も消えずに渦を巻き続けています。

こうした地球規模、あるいは惑星規模の大きな流れには、地球の自転と「ポテンシャル渦度」という概念が大きく影響しているんです。

今日は、地球の自転と風や海の流れがどのように結びついているのか、お話しします。

### §2. コリオリ力とは何か

#### メリーゴーランドでボールを投げると？

メリーゴーランドに乗りながら、隣にいる友達にボールを投げることを想像してみてください。
あなたはまっすぐ投げたつもりなのに、ボールは曲がって飛んでいきます。

急に野球の投手としての才能が目覚め、変化球の名手になったのでしょうか？
残念ながらそうではないんですね。

実は、メリーゴーランドの外から見れば、ボールはまっすぐ飛んでいるんです。
でも、あなたがボールを投げてから届くまでの間に、メリーゴーランドが回転しますよね。
だから、一緒に回っている友達の位置がずれてしまう。

あなたはメリーゴーランドと一緒に回っているので、自分が回転していることには気づきません。
だからボールのほうが曲がったように見えるんです。

```latex
% Manim アニメーションで表示
上からの視点: メリーゴーランドが反時計回りに回転
外部視点（慣性系）: ボールは直線に飛ぶ
回転視点（回転系）: ボールが右に曲がる軌跡を描く
2つの視点の対比アニメーション
```

このように、回転している世界にいると、実際には存在しない力を感じます。
この見かけ上の力を「コリオリ力」と呼びます。

```latex
% テロップで補足
コリオリ力 = 回転座標系に現れる見かけの力
Coriolis force: an apparent force in a rotating frame
```

#### 地球でも同じことが起きている

地球もメリーゴーランドのように回転しています。
私たちは地球と一緒に回転しているので、地球が回っていることを普段は意識しませんけど、超巨大なメリーゴーランドに乗っているようなものなんです。

でも、風や海流のように、地球の表面を長い距離移動する流体は、メリーゴーランドのボールと同じように、コリオリ力の影響をはっきりと受けるんです。

では、地球の上でコリオリ力がどう現れるか、もう少し具体的に見てみましょう。

地球は1日に1回自転しています。
このとき、地球の表面の速さは場所によって全然違うんです。

赤道では地球の円周がいちばん大きいので、地表は時速約1700キロという猛スピードで東に動いています。
でも緯度が上がるにつれて円周が小さくなるので、地表での速さはどんどん遅くなります。
そして北極点や南極点ではゼロ。その場でぐるぐる回っているだけになるんですね。

```latex
% Manim アニメーションで表示
地球の断面図: 赤道（時速約1700km）→ 北緯35°（時速約1400km）→ 北極（0km）
各緯度での地表の東向き速度ベクトルを矢印で表示
v = rω
円周上の速度 = 半径 × 角速度
```

ここで、赤道付近の空気が北に向かって移動する場合を考えてみましょう。

この空気はもともと赤道にいたので、時速約1700キロの東向きの速度を持っています。
北に移動しても、この東向きの速度はそのまま残るんです。

ところが、移動先の地面は赤道より遅い速さで東に動いている。
だから空気は地面に対して「東に逸れて」見えるんです。
つまり、北半球では北向きに動く空気は右に曲がるように見える。

```latex
% Manim アニメーションで表示
地球の平面図（北半球）:
赤道から北向きに移動する空気の矢印
空気が持つ東向き速度 > 移動先の地表速度
→ 空気が東に逸れる（右に曲がる）様子
```

じゃあ逆に、北のほうから赤道方向に動く空気はどうでしょう。

この空気はもともと高緯度にいたので、遅い東向きの速度しか持っていません。
南に移動すると、移動先の地面のほうが速く東に動いている。
だから今度は空気は地面に対して「西に逸れて」見える。
やっぱり右に曲がるんですね。

```latex
% Manim アニメーションで表示
地球の平面図（北半球）:
高緯度から赤道方向に移動する空気の矢印
空気が持つ東向き速度 < 移動先の地表速度
→ 空気が西に逸れる（右に曲がる）様子
```

つまり北半球では、空気がどちらに動いても進行方向に対して右に曲げられる。
南半球ではこれが逆になって、左に曲げられるんです。

```latex
% テロップで補足
北半球: 進行方向の右に曲がる
南半球: 進行方向の左に曲がる
NH: deflected to the right / SH: deflected to the left
```

#### コリオリパラメーター

このコリオリ力の強さは緯度によって変わります。
その強さを表す量を「コリオリパラメーター」と呼びます。

```latex
% Manim アニメーションで表示
f = 2Ω sin φ
コリオリパラメーター = 2 × 自転の角速度 × sin(緯度)
```

先ほど見たように、赤道では地表の速度差が南北であまり生じないので、コリオリ力は弱く、コリオリパラメーターはゼロになります。
一方、北極と南極では自転の効果がフルにかかるので、コリオリパラメーターは最大になるんです。

つまり、コリオリパラメーターは赤道でゼロ、北極と南極で最大。
北半球では南から北に行くほどコリオリパラメーターが大きくなっていきます。

```latex
% Manim アニメーションで表示
地球の模式図: 赤道(f=0) → 中緯度 → 極(f=最大) のグラデーション
```

このコリオリパラメーターが緯度によって変わることが、地球規模の風や海流の秘密を解き明かす鍵になるんです。

### §3. ポテンシャル渦度 — f と水深の関係

さて、コリオリ力を理解したところで、もうひとつの主役を紹介します。
それは「水深」です。

海の流れや大気の流れには、もちろん厚みがあります。
海なら海面から海底までの水深、大気なら対流圏の厚さですね。

実は、摩擦のない流体では「コリオリパラメーターと水深の比」が保存されるんです。
この比のことを「ポテンシャル渦度」と呼びます。
英語ではPotential Vorticity、略してPVですね。

```latex
% Manim アニメーションで表示
PV = f / h
```

```latex
% テロップで補足
厳密には PV = (ζ + f) / h
ζ: 流体自身の回転（渦度）
地球スケールでは ζ << f なので PV ≈ f / h
```

厳密には渦度と呼ばれる流体自身がその場で持っている回転の強さとコリオリパラメーターを足した値が分子になります。
ただし地球スケールの流れでは、渦度は地球の自転の効果に比べてとても小さいんです。
だから今回はこの渦度を無視して、PVはコリオリパラメーターと水深の比として考えていきます。

```latex
% テロップで補足
海洋での典型値: 渦度 ζ ≈ 1 cycle/月, コリオリ f ≈ 1 cycle/日
Typical values in oceans: ζ ≈ 1 cycle/month, f ≈ 1 cycle/day
```

この式が意味しているのは、こういうことです。

もし水深が変わったら、コリオリパラメーターもそれに見合うように変わらなければならない。
コリオリパラメーターが変わるということは、緯度が変わるということ。

つまり、PVの保存は「水深の変化が、流れの緯度方向の動きを決める」ということを意味しているんです。

```latex
% Manim アニメーションで表示
PV = コリオリパラメーター / 水深 = 一定
水深 が変わる → コリオリパラメーター も変わる → 緯度が変わる
水深の変化が流れの南北方向の動きを強制する
```

逆に、緯度が変わってコリオリパラメーターが変われば、水深のほうが変わることもありえます。

そしてこれは水だけの話ではありません。
大気でも同じで、大気の厚みが水深と同じ役割を果たすんです。

では具体的に、このPVの保存則がどんな現象を説明できるのか、見ていきましょう。

### §4. PV保存の威力

#### 例1: なぜ風は帯状に吹くのか

偏西風のような大気の大きな流れでは、大気の厚みがほぼ一定だと考えることができます。

厚みが一定の時、PVが保存されるためには、コリオリパラメーターも一定でなければなりません。
コリオリパラメーターが一定ということは、緯度が変わらないということ。
つまり、流れは南北方向にはほとんど動けず、東西方向に制限されるんです。

これが、偏西風や貿易風のような大気の大規模な流れが東西方向に帯状に吹く理由のひとつなんですね。


#### 例2: 海嶺が流れを曲げる

次に、東西に流れている海流が、海嶺、つまり海底の山にぶつかる場合を考えましょう。

海嶺の上では海底が盛り上がっているので、水深が小さくなります。
PVを保存するためには、コリオリパラメーターも小さくならなければなりません。
つまり、緯度が小さくなるように赤道方向に流れが曲がることになります。

```latex
% Manim アニメーションで表示
PV = f / h = const
h ↓ → f ↓
```

つまり北半球では、海嶺を越えようとする流れは南に曲げられるんです。

```latex
% Manim アニメーションで表示
海嶺を越える流れの模式図（断面図と平面図）
北半球で東向きの流れが海嶺にぶつかり、南に偏向する様子
専門的にはtopographic steeringと呼ぶ
```

逆に海底の谷である海溝、つまり水深が深くなる場所では、
PVを保存するためには、コリオリパラメーターも大きくなる必要がある。
つまり、緯度が大きくなるように北極方向に流れが曲がることになります。
そして谷を越えると今度は南に曲がって進むように流れるんです。

```latex
% Manim アニメーションで表示
海溝を越える流れの模式図（断面図と平面図）
北半球で東向きの流れが海溝にぶつかり、北に偏向する様子
```

```latex
% Manim アニメーションで表示
PV = f / h = const
h ↑ → f ↑
```

これをもっと大胆にいうと、コリオリパラメーターと水深の比が同じ場所を地図に線で書いたとき、海の流れはその線に沿って流れる、と言い換えることができます。

このような海底の地形によって海の流れが変わることの代表例が黒潮の蛇行です。
伊豆半島から小笠原諸島を通ってグアムやサイパンを含むマリアナ諸島にまで伸びる水深が浅くなっている伊豆・小笠原・マリアナ島弧を黒潮が越えて流れるとき、黒潮はPVを保存するために南に曲がって進みます。
これによって黒潮はグネグネ曲がった流れになるわけです。

実際には水温や気圧などの様々な要素が絡まり合って黒潮の流れは生み出されているのですが、海底の地形は黒潮の蛇行に最も大きな影響を与えている要因なんです。

さらに面白いのは、海嶺が高かったり、海溝が深くて、水深の変化が大きすぎる場合です。
緯度の変化だけではPVを保存しきれず、流れは海嶺や海溝を越えることができなくなるんですね。
結果として流れは海嶺や海溝に沿った動きをします。
このとき、海嶺や海溝はまるで壁のような役割になるんですね。

```latex
% Manim アニメーションで表示
PV = f / h = const
hの変化が大きすぎるとfの変化で対応できない
専門的にはtopographic steeringと呼ぶ
```

沖縄トラフがその代表例で、黒潮は沖縄トラフを越えることができずに九州の方へ流れているんです。

海底の地形が海の流れの方向を決めているのも、PVの保存から理解できる、とても面白い例なんですね。
