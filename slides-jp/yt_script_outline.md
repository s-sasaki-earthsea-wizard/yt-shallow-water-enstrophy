# 渦の保存則 — 2次元理想流体から浅水波方程式へ

## アウトライン

### 1. フック（イントロダクション）〜30秒

- かき混ぜた紅茶の茶葉、台風の衛星画像、洗面台の排水 — 日常のあちこちで「渦」を目にする
- この渦の流れを突き詰めていくと、地球の大気や海の流れ、木星の大赤斑を記述する理論に行きつく
- 今日は渦の物理についてお話しします

### 2. 渦度とは何か

- 流れの中に小さな正方形を置いて、その辺に沿って速度を一周分足し合わせる（線積分＝循環 Γ）
- この循環を正方形の面積で割った量が「渦度」ω
- 渦度は流体の各点での局所的な回転の強さを表す
- 2次元では ω = ∂v/∂x − ∂u/∂y というスカラー量になる

### 3. 理想流体とエンストロフィ保存

- 理想流体とは粘性がない流体。プールで前に進むと水が押し返してくるが、そのような抵抗がない理想的な世界
- このような流体では、大きな渦が小さな渦に分裂しても「渦の濃さ」の総量は変わらない
- この渦の濃さを表す量がエンストロフィ Z = ½∫ω² dA
- ωは∇×uで定義される。簡単に言えば小さな正方形に働く回転の力

### 4. 水面が動く現実 — 浅水波方程式

- ここまでは水面の高さが一定の、いわば「蓋のある水槽」のような2次元流体を考えてきた
- でも実際の海やプールでは水面が上下する — 波が立つ
- 水深に比べて水平方向のスケールが十分大きいとき、「浅水波近似」が使える
- 浅水波方程式は2つの式からなる：
  - 連続の式（質量保存）: ∂h/∂t + ∇·(h**u**) = 0 — 水の量は増えも減りもしない
  - 運動量方程式: ∂**u**/∂t + (**u**·∇)**u** = −g∇h — 水面の傾きが流れを駆動する
- 水深hが変わるということは、渦の「入れ物」の厚みが変わるということ
- では、§3で見た渦やエンストロフィの保存はどうなる？

### 5. ポテンシャル渦度

- 水深が変わると渦度ωそのものはもう保存しない
- 代わりに「渦度を水深で割った量」が保存される — これがポテンシャル渦度(PV, potential vorticity)
- PV = (ζ + f) / h　ここでζは相対渦度、fはコリオリパラメータ、hは水深
- 保存則: D/Dt [(ζ + f) / h] = 0（摩擦なしの場合）。D/Dtはラグランジュ微分を表す
- 直感的な意味：回転するピザ生地を想像する。生地を薄く伸ばす（h↓）と回転が弱まり（ζ↓）、厚くまとめる（h↑）と回転が強まる（ζ↑）
- コリオリ力fが一定（同じ緯度）なら、水深hが大きい場所ほど相対渦度ζも大きくなる
- 流体が南から北に移動するとfが増えるので、hが一定なら相対渦度ζは減少する
- 水を例にしたが空気でも基本的に同じ — 大気の厚みがhの役割を果たす
- この保存則はスウェーデンの気象学者ロスビーによって見出され、地球規模の風や海流の理解に大きな影響を与えた
- 現在、ロスビー波は気象予報や海洋予測で重要な役割を果たしている

### 6. まとめ 〜30秒

- 2次元理想流体では渦度ωとエンストロフィZがそのまま保存される
- 水面が動く浅水波の世界では、渦度をそのまま保存するのではなく、水深で割ったポテンシャル渦度 (ζ+f)/h が保存量になる
- 次元が変わっても、保存則は形を変えて生き残る — これが流体力学の美しさ
