# なぜ風は帯状に吹くのか — コリオリ力とポテンシャル渦度

## アウトライン

### 1. フック（イントロダクション）〜30秒

- 偏西風は西から東へ帯状に吹く、台風は北半球で反時計回り、木星の大赤斑は300年以上渦を巻き続けている
- これらの地球・惑星規模の流れには地球の自転と「ポテンシャル渦度」が影響している
- 地球の自転と風や海の流れがどう結びついているかを解説する

### 2. コリオリ力とは何か

#### メリーゴーランドでボールを投げると？

- 隣にいる友達にまっすぐ投げたのに、ボールは曲がって飛んでいく
- 野球の投手の才能が目覚めた？→ 残念ながら違う
- なぜ？ — 外から見ればボールはまっすぐ飛んでいる
- メリーゴーランドが回転する間に友達の位置がずれる
- 回転している自分には気づかないので、ボールが曲がったように見える
- この見かけ上の力が「コリオリ力」

#### 地球でも同じことが起きている

- 地球は超巨大なメリーゴーランドのようなもの
- 地球の表面速度は緯度で大きく異なる（v = rω）
  - 赤道: 時速約1700km（円周が最大）
  - 北緯35°: 時速約1400km
  - 極: 0km（その場で回るだけ）
- 赤道付近の空気が北に移動する場合:
  - もともと速い東向き速度を持っている → 移動先の地面より速い → 東に逸れる → 右に曲がる
- 高緯度の空気が赤道方向に移動する場合:
  - もともと遅い東向き速度 → 移動先の地面より遅い → 西に逸れる → やはり右に曲がる
- 北半球: 進行方向の右に曲がる / 南半球: 左に曲がる

#### コリオリパラメーター

- f = 2Ω sin φ（Ω: 自転角速度、φ: 緯度）
- 赤道: f = 0、極: f = 最大
- 北半球では南→北でfが増加
- このfの緯度依存性が地球規模の流れを理解する鍵

### 3. ポテンシャル渦度 — f と水深の関係

- 海の流れや大気の流れには厚み（水深h、大気の厚さ）がある
- 摩擦のない流体ではコリオリパラメーターfと水深hの比が保存される
- この比がポテンシャル渦度(PV): PV = f / h
- 厳密には PV = (ζ + f) / h（ζ: 渦度 = 流体自身の回転）だが、地球スケールでは ζ << f なので無視
  - 典型値: ζ ≈ 1 cycle/月, f ≈ 1 cycle/日
- PV保存の意味: hが変わる → fも変わる → 緯度が変わる（水深の変化が流れの南北移動を強制する）
- 逆にfが変わればhも変わる
- 水だけでなく大気でも同じ — 大気の厚みがhの役割

### 4. PV保存の威力

#### 例1: なぜ風は帯状に吹くのか

- h一定 → PV保存にはf一定 → 緯度が変わらない → 東西方向に流れが制限
- 偏西風・貿易風の帯状構造の説明

#### 例2: 海底地形が流れを曲げる

- 海嶺でh↓ → f↓ → 北半球では赤道方向（南）に偏向
- 海溝でh↑ → f↑ → 北半球では極方向（北）に偏向、谷を越えると南に戻る
- f/hが同じ場所を線で結ぶと、海の流れはその線に沿って流れる
- 黒潮の蛇行: 伊豆・小笠原・マリアナ島弧（水深が浅い）を越える際に南に偏向
- 水深変化が大きすぎると越えられない → 海嶺や海溝が壁の役割（topographic steering）
- 沖縄トラフ: 黒潮が越えられず九州方向へ流れる

#### ロスビーの貢献

- PV保存やコリオリパラメーターを通じた理論は1930〜40年代にカール＝グスタフ・ロスビーが確立
- 数理的気象学のパイオニア、コンピューター気象研究の最初期の人物

#### 木星の大赤斑

- ロスビーが提案したPV保存は地球以外の天体にも当てはまる
- PV保存の発展理論で大赤斑の理解が進む

### 5. まとめ 〜30秒

- コリオリ力や地形が地球上の流れに与える影響を概観
- 「PVが保存される」というシンプルな法則で地球スケールの流れを理解できる
- 身近な地形や風から新しい発見があるかもしれない
